# 🌐 Portfolio — Angular 20

A modern, responsive, and accessible single-page portfolio application built with **Angular 20** and deployed on **GitHub Pages**.  
Designed to showcase projects, skills, and experience with a clean UI, dark/light theme support, and multilingual support.

---

## ✨ Features

- **⚡ Angular 20** — latest standalone API & modern build system
- **🎨 Responsive UI** — built with SCSS & TailwindCSS
- **🌙 Light/Dark Theme** — automatic theme detection & manual toggle
- **🌍 i18n Support** — translations with `@ngx-translate/core`
- **📱 Mobile Friendly** — optimized for all devices
- **🚀 GitHub Pages Deployment** — CI/CD with GitHub Actions
- **♿ Accessibility** — WCAG-compliant forms and navigation
